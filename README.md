# Projeto Revisão

- Aplicativo mobile desenvolvido como atividade da disciplina de Programação para Dispositivos Móveis.

## Sobre o projeto

O projeto consiste em um aplicativo Android que reúne diferentes exercícios.

O objetivo principal é praticar:

- Navegação entre telas (Activities)
- Uso de Intents
- Passagem de parâmetros entre telas
- Estruturas condicionais
- Manipulação de componentes visuais

## Escopo

- O aplicativo possui as seguintes funcionalidades:

- Menu Principal
  - Tela inicial com navegação para todas as funcionalidades
- Mega-Sena
  - Geração de 6 números aleatórios
- IMC
  - Cálculo do índice de massa corporal
  - Exibição do resultado em diferentes telas
- Jokenpo
  - Jogo contra o computador
  - Sistema de placar (melhor de 3)
- Perfil
    - Cadastro simples de usuário
- Questionário da Felicidade
  - Cálculo baseado em sono e estresse
  - Exibição da pontuação e classificação

## Execução

- Instale o projeto com npm:

```bash
  npm install projetoFinalPDMII
  cd projetoFinalPDMII
  npm start
```

1. Abrir o projeto no Android Studio
2. Aguardar o carregamento do Gradle
3. Executar em um emulador ou dispositivo físico
